
# How to Set up Cobol
A brief description of set up onLinux and Windows


## Linux

- Install GNU Cobol compiler, instal IDE
- Edit program, debug, save
- Compile in bash : cobc -x ./file_name.cbl   /compile main program


## Windows

- Install OpenCobolIDE
- Edit program
- Debug in IDE
- Save
